Example: 'class' and 'v-bind:class'

La clase 'impClass' se configura de la forma normal con el atributo class y la clase 'yelClass' se configura con v-bind:class. El resultado es que Vue fusiona las clases para que el elemento div pertenezca a ambas clases.

This div belongs to both 'impClass' and 'yelClass'.
